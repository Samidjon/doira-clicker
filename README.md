# 🥁 Doira Clicker

An educational web-based clicker game designed to introduce players to the traditional Uzbek musical instrument **Doira** through interactive gameplay.

The game combines learning, gamification, quizzes, achievements, and an AI-inspired virtual assistant to make cultural education more engaging.

---

## 🎮 Features

### 🥁 Doira Clicker

Click the Doira to earn points.

- Real drum sound
- Animated click effects
- Random **+10 Bonus**
- Floating score animation

---

### 💡 Unlock Facts

As your score increases, Daniel unlocks interesting facts about the Doira.

| Points | Reward |
|-------:|--------|
| 50 | First Fact |
| 100 | Golden Doira |
| 200 | New Cultural Fact |
| 300 | Ancient Doira |
| 500 | Doira Master |

---

### 🏆 Ranking System

Earn points and increase your rank.

- 🌱 New Learner
- 🎵 Beginner Performer
- ⭐ Skilled Musician
- 🥁 Doira Master

---

### 👤 Player Profile

The profile page displays:

- Player name
- Total score
- Current rank
- Selected Doira
- Collection progress
- Achievements
- Facts unlocked

---

### 🥁 Doira Collection

Unlock different Doira designs.

- Classic Doira
- Golden Doira
- Ancient Doira

Each Doira can be selected after unlocking.

---

### 📝 Interactive Quiz

Daniel includes a built-in quiz.

- 15 questions
- +1 point for every correct answer
- Automatic score calculation
- Results are added to the main game score

---

### 💬 Daniel Assistant

The game includes an integrated chatbot.

Daniel can answer questions about:

- What is Doira?
- History
- Structure
- Playing techniques
- Uzbek culture
- Current score
- Achievements
- Unlockable Doiras
- Game mechanics

---

### 💾 Progress Saving

Game progress is automatically saved using **Local Storage**.

The following information is stored:

- Username
- Total score
- Selected Doira
- Current rank
- Quiz completion
- Unlockable items

Progress is restored automatically when the player returns.
